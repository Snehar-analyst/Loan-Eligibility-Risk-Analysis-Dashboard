# Loan-Eligibility-Risk-Analysis-Dashboard

## Project Overview

This project presents an interactive Loan Eligibility & Risk Analysis Dashboard built in Microsoft Excel. The dashboard analyzes loan applicant data to identify trends in income, loan amounts, credit history, education level, and property area. The objective is to provide insights that support loan approval and risk assessment decisions.

## Objectives

* Analyze applicant demographics and financial profiles.
* Identify patterns in loan amounts and household income.
* Evaluate the impact of credit history on loan applications.
* Compare applicant distribution across urban, semi-urban, and rural areas.
* Create an interactive dashboard for data-driven decision-making.

## Tools Used

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Slicers
* Conditional Formatting
* Excel Formulas

## Key Performance Indicators (KPIs)

* Total Applications
* Total Loan Amount
* Average Loan Amount
* Average Household Income
* Good Credit History Percentage
* Urban Applicants Percentage

## Dashboard Features

* Property Area Analysis
* Education-wise Loan Analysis
* Credit History Distribution
* Income Analysis
* Dependents Analysis
* Interactive Filters using Slicers

## Key Insights

* Applicants with a good credit history represent the majority of loan applications.
* Urban areas contribute a significant share of total applications.
* Higher household income is generally associated with larger loan amounts.
* Graduate applicants tend to apply for higher loan amounts compared to non-graduates.
* Most loans are issued with long-term repayment periods.
